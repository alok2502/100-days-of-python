## Topic: Functions with Parameters

## What I Learned Today

### Parameters vs Arguments
Learned the difference between **parameters** and **arguments** in Python functions.

- **Parameter**: The variable name defined in the function
- **Argument**: The actual value passed to the function when calling it

Understanding this difference helped clarify how functions receive and use data.

### Types of Arguments

Learned about two main types of arguments:

#### Positional Arguments
- Values are passed in the same order as parameters
- Order matters while calling the function

#### Keyword Arguments
- Values are passed using parameter names
- Order does not matter
- Makes function calls more readable and less error-prone
